Sistema Embebido

Se creó un sistema embebido que realiza las tareas de sensado y envío de porcentaje de temperatura y humedad dentro del microtúnel

Wi-Fi

Uso de conexiones inalámbricas para el envío de datos.

Graficos

La aplicación obtiene los datos de internet y los muestra en una gráfica lo que facilita el entendimiento de los datos generados por el sistema embebido

Tablas

Tablas que muestran el historial de temperatura, humedad, fecha y hora en el microtúnel.
